# Intro to Tanzu Platform Workshop

This repository contains workshop materials for introducing developers to Tanzu Platform for Cloud Foundry and the experience of building and deploying mission ready AI applications on the platform.

The workshop is designed as a hands-on introduction to the Tanzu developer workflow. Participants will deploy an application with `cf push`, explore Cloud Foundry application operations, and progressively add AI capabilities through platform-managed services such as GenAI, embeddings, vector storage, MCP tools, and persistent memory.
