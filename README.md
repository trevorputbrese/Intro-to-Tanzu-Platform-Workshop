# Intro to Tanzu Platform Workshop

This repository contains workshop materials for introducing developers to Tanzu Platform for Cloud Foundry and the experience of building and deploying mission ready AI applications on the platform.

The workshop is designed as a hands-on introduction to the Tanzu developer workflow. Participants will use a sample AI enabled application and learn how Tanzu Platform can provide app teams the ability to rapidly prototype aps by going from code to running app in minutes.  Devs will learn application application operations, and progressively add AI capabilities through platform-managed services such as AI Servicess, vector storage, MCP tools, and other services offered via the Tanzu Marketplace.
